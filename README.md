# nav-gauge

Upload a `.gpx` file, choose styles and analyze a trip in a nav gauge component. 
Record with an external tool or save a series of pictures.

## Development

### Web app

#### Start development build

```
yarn --cwd apps dev
```

#### Production build
```
yarn --cwd apps build
yarn --cwd apps start
```
